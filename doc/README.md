## 1. 文档目标

本目录为 `Go2-X5 sim2real` 部署提供完整的设计和实施文档。

**核心范式**：
- PPO 只控制 12 个腿关节（locomotion）
- 机械臂由外部pipeline控制（arm独立）
- 系统重点是 arm-conditioned base stability
- Coordinator负责混合控制协调
